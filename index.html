How to Use Image Resizer

100% Free - Just register to get started. No hidden costs.

Quick Start Guide

  1. 1Upload an image - Drag & drop or click to browse
  2. 2Choose a size - Pick a preset or enter custom dimensions
  3. 3Adjust quality - Use the slider to balance size vs quality
  4. 4Download - Get your resized image

Preset Sizes

  • Social Media - LinkedIn, Facebook, Instagram, X (Twitter), YouTube
  • Official / ID - Passport photos (UK, US, EU, India, China), driving licence, CV
  • Web / Apps - Favicons, app icons, avatars, OG images, screen resolutions

Tip: Presets use the exact dimensions required by each platform.

Custom Sizes

  • Lock aspect ratio - Click the lock icon to maintain proportions
  • Enter dimensions - Type exact width and height in pixels
  • Format options - JPEG (smaller files), PNG (transparency), WebP (modern)
  • Quality slider - Higher = better quality but larger file size
